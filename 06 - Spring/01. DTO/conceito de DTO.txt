DTO - Data Transfer Object

1- Camada de segurança para não alterar informações sensíveis.
Ex. 
{
"codigo":1,
"nome": "Alice",
"email":"alice.silva@gmail.com",
"senha":"123",
"tipo":"comum"
}

2- Retorno de dados - características que serão retornadas ou não.
Ex.
{
"codigo":1,
"nome": "Alice",
"email":"alice.silva@gmail.com",
"tipo":"comum"
}
